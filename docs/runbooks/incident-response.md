# Incident Response Runbook

## Severity Levels
- SEV-1: Complete outage (15 min response)
- SEV-2: Major feature unavailable (30 min)
- SEV-3: Degraded performance (2 hours)
- SEV-4: Minor issue (24 hours)

## Steps: Acknowledge alert, open incident channel, assess severity, notify stakeholders, root cause analysis, apply mitigation, write post-mortem.
