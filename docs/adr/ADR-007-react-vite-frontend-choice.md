# ADR-007: React + Vite Frontend

Status: Accepted

## Decision
React 18 plus Vite plus TypeScript for the SPA.

## Rationale
- Fast HMR with Vite
- React concurrent features for enterprise scale
- TypeScript-first for enterprise codebase safety
