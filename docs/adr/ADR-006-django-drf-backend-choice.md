# ADR-006: Django + DRF Backend

Status: Accepted

## Decision
Django 4.x plus DRF for the API layer.

## Rationale
- Mature ORM with excellent PostgreSQL support
- Django admin for platform operations
- Celery integrates natively
- B01 canonises Django as the backend framework
